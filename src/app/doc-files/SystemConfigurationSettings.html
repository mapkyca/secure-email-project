System Configuration Settings

These are settings that can be used to configure various aspects of the email proxy system. These settings are loaded by the EmailProxy program on startup from the specified file.

Variable
Default value
 Description
proxyserver.outgoing.port
25
 The local port the proxy listens to for outgoing (SMTP) mail.
proxyserver.incoming.port
110
 The local port the proxy listens to for incoming mail.
proxyserver.auth.require
1
 Should the proxy server require an SMTP Auth logon (recommended). "1" for yes "0" for no.
proxyserver.auth.username
undefined
 The username the proxy server will expect.
proxyserver.auth.passcode
undefined
 An MD5 hash of the auth passcode.
 
algorithm
openpgp
 What algorithm should be used for encryption and signing (used for future expansion). This also governs the dialog class used for key management.

keymanager.openpgp.publiclist.<n>.type
The type of key handler, where <n> represents the priority. The higher the number the lower the priority. Note, no matter how high a priority you specify you will never supercede the primary key handlers.

Type is the class that controls this type. Note, that for security reasons the class can only be loaded from the core.keyhandler package, and is prefixed with "core.keyhandler."

Classes do not need the ".class" extension, this is assumed.
keymanager.openpgp.secretlist.<n>.type

The type of key handler, where <n> represents the priority. The higher the number the lower the priority. Note, no matter how high a priority you specify you will never supercede the primary key handlers.

Type is the class that controls this type. Note, that for security reasons the class can only be loaded from the core.keyhandler package, and is prefixed with "core.keyhandler."

Classes do not need the ".class" extension, this is assumed.
keymanager.openpgp.<list>.<n>.filename
The the key handler entry is a file type, this is the filename. Where <n> is the priority, and <list> is either secretlist or publiclist.
keymanager.openpgp.<list>.<n>.serveraddress
If this is a server, the address of the server.
keymanager.openpgp.<list>.<n>.serverport
If this is a server, the port on the server to connect to.
